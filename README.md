# ketok-id Homebrew tap

```sh
brew tap ketok-id/tap
brew install --cask murmur
```

[Murmur](https://murmur.ketok.id) — a menu-bar player for YouTube audio,
internet radio (58k stations), live TV, and the 2026 World Cup, for macOS 13+.

Updating: `brew upgrade --cask murmur`. The app also self-updates in place.
